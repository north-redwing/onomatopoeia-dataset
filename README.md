# Onomatopoeia Dataset
The Onomatopoeia Dataset is labeled audio forked from [ESC-50](https://github.com/karolpiczak/ESC-50).  

## Labels
The labels are japanese onomatopoeia words given by an author.  
- sheep: me- (メー)
- cat: nya- (ニャー)
- dog: wan (ワン)
- longcarhorn: pu- (プー)
- shortcarhorn: pu (プッ)
- churchbells: ka-nn (カーン)
- crow: ka- (カー)
- creaks: gi- (ギー)
- chainsaw: ui-nn (ウィーン)
- breathing: ha- (ハー)
